## How to use: 
1. Install **Node.js** from https://nodejs.org/en/ if you don't have it installed already.
2. Clone this repo or download this as a zip and open the folder.
3. Open terminal in that folder/directory and bash:-
```sh
npm init -y
```
```sh
npm i discord-rpc
```
4. To run it bash:-
```sh
node index.js
```
